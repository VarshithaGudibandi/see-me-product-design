## Overview

Designed and specified a premium subscription tier connecting artists and buyers through exclusive prints and live events.

This project focused on building a scalable monetization layer while preserving marketplace dynamics.

---

## Problem

Art buyers lacked structured access to exclusive content and deeper engagement with artists.  
The platform lacked a recurring revenue model and premium gating infrastructure.

---

## Objectives

- Introduce a subscription revenue stream  
- Increase buyer engagement and retention  
- Enable artists to opt into premium experiences  
- Design scalable access control architecture  

---

## Approach

- Conducted user segmentation and willingness-to-pay analysis  
- Prioritized features using importance vs effort framework  
- Designed gated subscriber flows for prints and live events  
- Defined KPIs for adoption, retention, and AOV  
- Wrote detailed PRD with functional and non-functional requirements  

---

## Key Features

- Exclusive artist print drops  
- Premium event access  
- Artist opt-in subscription controls  
- Tier-based gated content  
- Monetization KPI framework  

---

## Metrics Defined

- Premium adoption rate  
- Subscription churn  
- Average order value (AOV)  
- Subscriber engagement rate
